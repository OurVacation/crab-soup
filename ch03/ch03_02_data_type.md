## 1. 🥘 핵심 재료 (Key Concepts) : 데이터 타입

Rust의 모든 값은 특정 데이터 타입을 가지며 이는 스칼라 타입 / 복합 타입 으로 나뉜다.

러스트는 정적 타입 언어다. 이 말은 컴파일 시점에 모든 변수의 타입이 반드시 지정되어 있어야 한다는 뜻이다.

### (1) 스칼라 타입
- **정의:** 항상 불변인 값
- **종류 및 특징**
    1. 정수형 : 소수점 없는 숫자, 그런데 Rust는 특이하게 비트 크기까지 붙여서 정수를 아래처럼 표시한다.
        | 길이 (bit) | signed | unsigned |
        | --- | --- | --- |
        | 8 | i8 | u8 |
        | 16 | i16 | u16 |
        | 32 | i32 | u32 |
        | 64 | i64 | u64 |
        | 128 | i128 | u128 |
        | arch | isize | usize |
    2. 부동 소수점 : 정수와 동일하게 `f32`와 `f64` 두 개의 기본 타입을 갖는다
    3. 부울린(boolean) : 특별한거 없음
    4. 문자(char) : 크기는 4byte이며, 유니코드 스칼라 값을 따르기 때문에 이모티콘, 중국어, 한자 등등의 별의별 문자도 전부 다 표현 가능하다

### (2) 복합타입
- **정의:** 여러 값을 하나로 묶은 것
- **종류 및 특징**
    1. 튜플
        : 길이가 고정되어 있으며, 한번 선언되면 크기 조절 불가능
        ```rust
        let tup: (i32, f64, u8) = (500, 6.4 , 1);
        // 튜플 개별 값을 꺼내오려면 아래처럼 하면 됨
        // 이를 구조해체라 한다
        let (x, y, z) = tup;

        println!("The value of y is {y}");
        // y에 해당하는 64.1 출력

        // 튜플로 선언된 변수 뒤에 .으로 접근 가능
        let five_hundred = tup.0;
        let six_poiny_four = tup.1;
        ```

        **지식 한 줌** : 빈 튜플 `()` 은 유닛이라 표현되며, 빈 값, 또는 빈 반환 타입을 나타낸다. 표현식의 반환값이 없다면 암묵적으로 유닛값을 반환한다.

	2. 배열
        : 길이와 타입이 고정되며, 스택에 메모리가 할당된다.
        ```rust
        // let 배열 이름 [타입; 길이] = [...]
        let a[i32 ; 5] = [1,2,3,4,5];
        // 따로 타입 지정 없이도 사용 가능
        let months = [Jan, Feb, Mar, Apl, May, Jun, Jul,
                                            Aug, Sep, Oct, Nov, Dec];
        ```
        런타임 중 배열 길이 초과하면 에러 발생시키고 종료. 메모리 접근 허용하지 않음

## 3. 🧂 깊은 맛 (Deep Dive)
- 유닛은 `NULL` 일까?
    아니다! 대신 값이 없을 가능성은 Option 열거형으로 안전하게 다룬다. C의 NULL은 (void*)0으로 정의되어 메모리 주소 0을 가리키지만, 유닛은 주소값이 아니라 "의미 있는 값이 없음"을 나타내는 하나의 구체적인 값이다.

    따라서 유닛을 참조하더라도 그것은 유효한 유닛 타입의 인스턴스를 가리키는 것이지, NULL처럼 유효하지 않은 메모리 주소를 건드려 프로그램을 터뜨리는 일은 발생할 수 없다.
